# Security Policy

## Supported Versions

This repository is research software, not a production service. Security support is
best-effort and focused on the current `main` branch and the latest tagged release.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | Yes                |
| latest tagged release | Yes |
| older releases | Best effort |

## Reporting a Vulnerability

Please report security issues by email to `dr.sayujkrishnan@gmail.com` or through the
GitHub Security tab if one is available for the repository.

Include:

* A short description of the issue.
* The file or feature affected.
* Steps to reproduce, if applicable.

We will acknowledge receipt as soon as practical and follow up with any requested
mitigation or disclosure steps.
