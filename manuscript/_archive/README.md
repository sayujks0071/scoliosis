# Archived manuscript drafts

This directory contains **legacy manuscript drafts and stub files** that are *not* part of the current Spine Deformity submission. They are preserved for revision history only.

The active submission is `manuscript/main.tex` (with sections in `manuscript/sections/`). Nothing in this directory is `\input` by `main.tex`.

## Why archived

These files predate the clinical-language reframe (v1.0 → v1.3) and either:
- target a different journal (Nature drafts), or
- retain earlier theoretical-physics vocabulary that does not match the current Spine Deformity submission's framing, or
- are unused TikZ figure stubs that were superseded by PDF figures in `manuscript/figures/`.
